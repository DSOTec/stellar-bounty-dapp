# Stellar Bounty DApp

A decentralized platform for creating, managing, and completing bounties
on-chain using Stellar Soroban smart contracts.

---

## 🌍 Overview

**Stellar Bounty DApp** is an open collaboration platform that enables
individuals, startups, and communities to post tasks (bounties) and reward
contributors transparently using blockchain technology.

Instead of relying on centralized platforms, this application leverages
**Stellar + Soroban** to ensure:

- Trustless payments
- Transparent task management
- Global accessibility

---

## ✨ Problem

Open collaboration is powerful, but current systems have limitations:

- ❌ Delayed or disputed payments
- ❌ Lack of transparency in task completion
- ❌ Centralized control over funds
- ❌ Limited access for global contributors

---

## 💡 Solution

Stellar Bounty DApp solves this by introducing:

- ✅ On-chain bounty creation
- ✅ Escrow-based payments
- ✅ Transparent submission & approval flow
- ✅ Instant payouts via Stellar network

---

## 🧩 Core Features

### 🔹 Bounty Management

- Create bounties with defined rewards
- Set requirements and deadlines
- Fund bounties securely on-chain

### 🔹 Contributor Workflow

- Browse available bounties
- Submit completed work
- Track submission status

### 🔹 Payment & Escrow

- Funds locked in smart contracts
- Release payments upon approval
- Secure and tamper-proof transactions

### 🔹 Wallet Integration

- Connect Stellar-compatible wallets
- Sign and approve transactions seamlessly

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/DSOTec/stellar-bounty-dapp.git
cd stellar-bounty-dapp
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App

```bash
npm run dev
```
